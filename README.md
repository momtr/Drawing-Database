# Drawing Database
This project has been created to collect data for training an AI. <br>
It uses Node.js and runs a server locally on port 3000. <br>

## Dependencies
- express
- body-parses
- dedb (for storing JSON objects)

## Front-End
Here we use p5.js for making the drawing.
